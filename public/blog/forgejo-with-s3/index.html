Running Forgejo with S3 storage.

2025-04-21 · Alex

The official forgejo alternate storage services guide only talks about minio so I thought I’d make an example for an actual s3 service.

Packages are huge nowadays, especially docker images which led me down the path of finding a viable solution for storing these for cheap in the cloud.

Config Example (Add to app.ini)

[storage]
STORAGE_TYPE = minio

MINIO_ENDPOINT = s3.ap-southeast-syd.cloud.ovh.net
MINIO_ACCESS_KEY_ID = [redacted]
MINIO_SECRET_ACCESS_KEY = [redacted]
MINIO_BUCKET = alecks-forgejo
MINIO_LOCATION = ap-southeast-syds
MINIO_USE_SSL = true

I’m using ovh object storage because my forgejo server is on an ovh vps where connections to their object storage count as “Internal traffic” and is completely free.
